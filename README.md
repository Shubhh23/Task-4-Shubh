# Data Visualization & Business Insights Project

## Project Objective
The objective of this project is to analyze e-commerce customer data and communicate meaningful business insights through effective data visualization techniques.

## Key Analyses Performed
- Product Demand Analysis
- Payment Method Preference Analysis
- Order Status Distribution Analysis
- Customer Acquisition Source Analysis
- Order Value Distribution Analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings
- Printer is the most demanded product.
- Online payment is the most preferred payment method.
- Instagram is the leading customer acquisition channel.
- Order cancellations require further investigation.
- Most customers place lower-value orders, while a few high-value orders contribute significantly to revenue.

## Outcome
The project demonstrates how data visualization can transform raw data into actionable business insights and support better business decisions.
